# Chaos Gate Unlocker

![Screenshot from 2025-04-03](https://staticdelivery.nexusmods.com/mods/4502/images/5/5-1739619171-398274387.png)

Easy-to-use tool that lets you edit game saves to improve your gaming experience.

[![GitHub Release](https://img.shields.io/github/v/release/imsgit/chaos-gate-unlocker)](https://github.com/imsgit/chaos-gate-unlocker/releases)

## About

This tool is designed to make the Dreadnought available more often by authorizing missions for him and making repairs easier. Another key feature is unlocking advanced classes from the beginning, since they appear too late in the game. The final major concept is to let players use the Gladius frigate from the start, adding a fresh dynamic to the game.

Players are expected to progress through the campaign, explore, collect gear, level up, and master the game at a steady pace, keeping the core experience of discovery and challenge. This tool doesn't allow altering in-game currencies, adding items or granting XP. It saves time by unlocking things earlier while still keeping the overall enjoyment intact.

## Prerequisites

You need to have the base game installed on your system. DLCs are optional and cannot be unlocked by this tool, buy them first.

This tool does not use or modify any game assets or binaries, nor does it inject anything into the game’s runtime. It also does not collect or send any data over the internet. All icons used in this tool were sourced from public resources and adapted accordingly.

Please note that this tool does not back up save files, so I recommend doing so manually. If you're using Steam, it's advisable to disable cloud saves.

This tool is available for Windows and Linux and is distributed free of charge. If you encounter a bug, please contact me.

## Getting Started

1. Open the tool and load a save with the **Open** button.
2. Switch on the options you want on the **Main** and **Units** pages.
3. Press **Save** to write the changes back, then continue playing in-game.

## Main page

The Main page is split into two columns. The left column holds Dreadnought and gear actions you can apply whenever you like, while the right column holds one-time unlocks.

<img src="https://staticdelivery.nexusmods.com/mods/4502/images/5/5-1781079112-102612344.png" alt="Main page" width="820">

**Left column**

- **Authorize Dreadnought missions** — marks all regular missions (including Hive missions) as Technophage so the Dreadnought can join them; mission difficulty rises and frigate missions are unaffected (Dreadnought access required).
- **Repair Dreadnought** — repairs the Dreadnought's damage so it returns to action sooner (Dreadnought access required).
- **Restore prognosticars** — makes all attuned prognosticars available again.
- **Complete current research** — finishes the active research project instantly (advance time to take effect).
- **Complete current construction** — finishes the active construction project instantly (advance time to take effect).
- **Unequip mastercrafted armor** — unequips all mastercrafted armor and frees the slots, also unlocking unavailable armor; frigate missions are unaffected.
- **Unequip mastercrafted weapons** — unequips all mastercrafted weapons, also unlocking unavailable weapons; frigate missions are unaffected.

**Right column — one-time unlocks**

- **Unlock pre-order items** — unlocks the Domina Liber Daemonica tome and Destroyer of Crys'yllix hammer.
- **Unlock advanced classes** — unlocks the Librarian, Paladin, Chaplain and Purifier classes (advance time to take effect).
- **Unlock Garran Crowe** — unlocks castellan Garran Crowe (DLC required; advance time to take effect).
- **Unlock assassins** — unlocks the imperial assassins (DLC required; advance time to take effect).
- **Unlock Gladius frigate** — unlocks the Gladius frigate; the Cleanse mission still appears as expected (DLC required; advance time to take effect).
- **Unlock purity seals** — unlocks purity seal upgrades (Poxus seeds required; advance time to take effect).
- **Remove marketing weapons** — unequips and removes all weapons classified as Twitch drops.

## Units page

On the Units page everything is organized in the same order as the units appear in the game. You can view the status and information of each unit and apply per-unit changes.

<img src="https://staticdelivery.nexusmods.com/mods/4502/images/5/5-1781079122-18312124.png" alt="Units page" width="820">

- **Augmetics** — when you heal a critical wound, you can add an augmetic or change existing ones. A knight cannot have more augmetics than the resilience they have lost.
- **Talents** — manage a unit's existing talents. Extra talents can be earned through special events, such as A Solitary Duty or The Tournament.
- **Retrain** — refunds all spent ability points so you can rebuild a unit's abilities from scratch. The unit keeps its level and any separately learned class perks.

## Known issues

You may still see in-game notifications for purity seals, advanced classes and frigate later in the game.
